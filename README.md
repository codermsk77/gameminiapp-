# Натренированный отдых — Паттерны

Telegram Mini App для тренировки концентрации через паттерны.

## Возможности

- **24 паттерна** — соединяйте точки в любом порядке
- **Режимы**: по порядку или случайный
- **Прогресс** — текст «5/24» и визуальная полоска
- **Кнопка перезапуска** в конце сессии
- **Справка** — кнопка «?» с описанием методики

## Разработка

```bash
npm install
npm run dev
```

Откройте в браузере или через [@BotFather](https://t.me/BotFather) → Bot Settings → Menu Button / Web App.

## Сборка

```bash
npm run build
```

Файлы в `dist/` можно развернуть на любом статическом хостинге (Vercel, Netlify, GitHub Pages и т.д.).

## Подключение к боту

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. В настройках бота укажите URL вашего Mini App
3. Добавьте кнопку меню или команду для открытия приложения
